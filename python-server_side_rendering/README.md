# Python - Server-Side Rendering

## Overview
Implementation of Server-Side Rendering (SSR) using Python, Flask, and Jinja2 templating. Includes dynamic data integration from JSON, CSV, and SQLite databases.

## Requirements
* **Language:** Python 3.
* **Framework:** Flask.
* **Templating Engine:** Jinja2.
* **Databases/Data Formats:** SQLite, JSON, CSV.

## Files
* `task_00_intro.py`: Basic string templating and file I/O operations.
* `task_01_jinja.py`: Basic Flask application with HTML templates and reusable components.
* `task_02_logic.py`: Dynamic templates utilizing Jinja loops, conditionals, and JSON parsing.
* `task_03_files.py`: URL query parameters handling and dynamic data loading (JSON/CSV).
* `task_04_db.py`: SQLite database integration and query execution for template rendering.
