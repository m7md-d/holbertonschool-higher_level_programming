# JavaScript - DOM Manipulation

## Overview
Implementation of DOM interaction, event handling, and asynchronous data fetching using ES6.

## Requirements
* **Language:** JavaScript (ES6).
* **Standards:** Semistandard compliant.
* **Constraints:** Use of `var` is prohibited; use `const` or `let`.
* **Behavior:** No page reloads allowed for DOM updates or data fetching.
* **Environment:** Chrome browser (version 57.0+).
