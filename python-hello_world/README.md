# Python Hello World

This directory contains introductory **Python exercises** focused on basic syntax, string manipulation, and formatted output.

The tasks are designed to practice simple Python concepts such as printing, variables, f-strings, slicing strings, and following strict code constraints.

---

## Files

### `2-print.py`

* Prints a simple sentence using the `print` function.

### `3-print_number.py`

* Prints an integer followed by text using **f-strings**.

### `4-print_float.py`

* Prints a floating-point number with **two decimal places**.

### `5-print_string.py`

* Prints a string multiple times.
* Displays the first part of the string using slicing.

### `6-concat.py`

* Concatenates strings to form a complete sentence.

### `7-edges.py`

* Extracts specific parts of a string using slicing:

  * First 3 characters
  * Last 2 characters
  * Middle part

### `8-concat_edges.py`

* Builds a sentence using string slicing.
* No new variables or string literals allowed.

### `9-easter_egg.py`

* Prints **“The Zen of Python”** by Tim Peters.

---

## Purpose

The purpose of this directory is to:

* Learn Python basics
* Practice string manipulation
* Apply formatting with f-strings
* Follow strict coding constraints
